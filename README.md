# 恋みくじアプリ (Koi-Mikuji App)

ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなウェブアプリケーションです。

## 概要

このプロジェクトは、HTML、CSS、JavaScriptの3つのファイルで構成されています。
ユーザーが「おみくじを引く」ボタンを押すと、あらかじめ用意された4種類の運勢（大吉、中吉、小吉、凶）からランダムに1つが画面に表示されます。

## ファイル構成

本プロジェクトは以下の3つのファイルで構成されています。

1. **index.html**
   - アプリの構造を定義しています。
   - タイトル、説明文、おみくじを引くためのボタン、結果表示エリアが含まれます。
2. **style.css**
   - アプリのデザイン（レイアウトや色使い）を定義しています。
   - 背景色を青色に、ボタンをピンク色に設定し、親しみやすいデザインにしています。
3. **script.js**
   - おみくじのロジック（動作）を担当しています。
   - ボタンがクリックされた際に、ランダムな数字を生成して運勢を決定し、画面を書き換えます。

## 使い方

1. 全てのファイル（`index.html`, `style.css`, `script.js`）を同じフォルダに保存します。
2. ブラウザで `index.html` を開きます。
3. 「おみくじを引く」ボタンをクリックすると、結果が表示されます。

## 技術的なポイント

- **ランダム機能**: `Math.random()` を使用して、配列の中からランダムに要素を取得しています。
- **イベントリスナー**: JavaScriptの `addEventListener` を使用して、ボタンクリックというユーザーの操作を検知しています。
- **DOM操作**: `textContent` を使用して、計算された結果をHTML上に動的に表示させています。

## ライセンス

このプロジェクトは学習用として自由に使用・改変が可能です。
